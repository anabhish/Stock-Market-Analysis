# Stock-Market-Analysis
This project focuses on analysis of the stock market based on the
following parameters:
1.	The change in price over time.
2.	Moving average of various stocks.
3.	Comparing adjusted closing values to analyze if there are any correlation between the stocks.
4.	Value at Risk by Comparing the Expected returns and the Standard Deviation of the annotation i.e, Risk.
5.	Value at risk using Monte carlo method
